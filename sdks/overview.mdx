## Software Development Kits

The Flux OS platform provides SDKs to help you integrate your data and systems efficiently. Below are the SDKs available for Python and .NET, along with instructions on how to get started.

### Python

The Flux OS Python SDK allows you to seamlessly interact with the Flux OS API using Python. This SDK provides pre-built methods to authenticate, retrieve data, and perform operations without having to write low-level code.

#### Installation

To install the Flux OS Python SDK, run the following command in your terminal:

```bash
pip install flux-os
```
